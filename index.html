Crescent library brings privacy to digital identity systems

Digital identities, while convenient, introduce significant privacy risks through "linkability," where hidden identifiers enable tracking of user behavior across various applications. To combat this, Microsoft Research has released Crescent, a cryptographic library that adds "unlinkability" to common identity formats like JSON Web Tokens and mobile driver's licenses. Crescent utilizes zero-knowledge proofs and preprocessing to allow users to prove facts about their credentials without revealing linking data, ensuring privacy and efficient operation on mobile devices without requiring system updates from issuing organizations.
