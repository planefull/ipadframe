RedCodeAgent: Automatic red-teaming agent against diverse code agents

Traditional safety benchmarks and red-teaming methods are inadequate for evaluating the real-world security risks of AI code agents, as they often fail to assess actual code execution behaviors or complex attack scenarios. To address this, researchers developed RedCodeAgent, an automated and adaptive red-teaming agent designed to evaluate large language model-based code agents. RedCodeAgent uses an adaptive memory, a specialized toolbox with code substitution, and simulated execution environments to efficiently uncover diverse vulnerabilities and outperform existing jailbreak methods by achieving higher attack success rates.

M5 Ultra Chip Coming to Mac Studio in 2026

Apple is reportedly planning to debut an M5 Ultra chip in 2026, which is expected to power the Mac Studio and a potential Mac Pro update. This M5 Ultra will likely follow the M5 Pro and Max variants, which are anticipated in early 2026. Notably, there will be no M4 Ultra, as the M4 Max chip lacks the necessary UltraFusion connector, confirming Apple's strategy of not releasing an Ultra variant for every M-series generation.
