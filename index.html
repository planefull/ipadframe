iOS 26.2 Adds Temporary AirDrop Access With Shareable Codes

iOS 26.2 beta 3 introduces a new AirDrop feature that allows users to temporarily share files with people not in their contacts. This functionality uses a shareable code, which provides 30-day AirDrop access for file sharing between devices. This new option offers a secure way to connect without adding contacts or broadly enabling AirDrop for everyone nearby.
