Crescent library brings privacy to digital identity systems

Digital identities, despite their convenience, introduce privacy risks through "linkability," where hidden identifiers allow separate uses of credentials to be tracked and associated with an individual, even with existing selective disclosure features. To address this, Microsoft Research has released Crescent, a cryptographic library that adds "unlinkability" to common identity formats like JSON Web Tokens and mobile driver's licenses. Crescent utilizes zero-knowledge proofs, optimized for mobile devices, to prevent linking separate digital interactions and protect user privacy without requiring issuers to update their systems.

How to Install iOS 26 Public Beta 6 on Your iPhone

Apple has released the sixth public beta of iOS 26, offering a more stable version of the software for a wider audience after extensive developer testing. The article provides instructions on how to enroll in Apple's free Beta Software Program, create an archived backup, and install the beta, which is compatible with iPhone 11 series and newer. Users are cautioned about potential bugs and advised to install it on a secondary device, highlighting new features like a "Liquid Glass" redesign, Apple Intelligence, and updated communication and core apps.
