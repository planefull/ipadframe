iOS 26.2 Adds These New Features to Your iPhone

iOS 26.2, currently in beta testing, is slated for release to all users in December, bringing numerous new features to iPhones 11 and newer. This update includes a new Liquid Glass slider for the Lock Screen clock, offline lyrics for Apple Music, revised Sleep Score ranges, and the ability to set alarms for Reminders. Additionally, it enhances Apple Podcasts and News, adds an iPhone screen flash for notifications, and expands AirPods Live Translation functionality to the EU.
