RedCodeAgent: Automatic red-teaming agent against diverse code agents

RedCodeAgent is a novel, fully automated and adaptive red-teaming agent designed to evaluate the safety of large language model-based code agents, specifically addressing the shortcomings of existing static benchmarks in assessing real-world execution risks. It operates with a memory module for continuous learning, a specialized toolbox including code substitution, and integrates simulated sandbox environments to assess the impact of generated code. This innovative approach consistently achieves higher attack success rates, lower rejection rates, and uncovers previously unknown vulnerabilities across diverse code agents and programming languages, significantly outperforming traditional jailbreak methods.

Nintendo Store App Launches on iPhone With a Feature Fans Will Love

Nintendo has launched a new "Nintendo Store" app for iPhone and iPad, enabling users to browse games, consoles, accessories, and merchandise, although purchases are directed to the company's web store. A popular feature allows users to sign in and review their play activity across Nintendo Switch, Switch 2, 3DS, and Wii U consoles. The free app also offers news, wish list notifications, and is available in various regions including the US, Canada, Mexico, Brazil, several European countries, and Japan.
