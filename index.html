Everything New in iOS 26.2 Beta 2

Apple has released the second developer beta for iOS 26.2, introducing several new features and design updates. Key changes include a "Liquid Glass" design for the Measure app's level, new sorting options for the Games app, and bouncier menu animations. Additionally, the beta provides more details on Enhanced Safety Alerts for emergencies like earthquakes or floods, which are enabled by default and offer improved delivery using approximate location.
