Crescent library brings privacy to digital identity systems

Digital identities, while convenient, introduce significant privacy risks through "linkability," allowing separate uses of credentials to be tracked and associated with individuals. To counter this, Microsoft Research has developed Crescent, a cryptographic library that integrates "unlinkability" into widely used identity formats such as JSON Web Tokens and mobile driver's licenses. Crescent leverages zero-knowledge proofs to enable users to selectively disclose information without revealing underlying data or linking their various digital interactions, enhancing privacy without requiring changes to existing issuing systems.

Apple Releases Xcode 26 Beta 7 With GPT-5 Support and Claude Integration

Apple has released Xcode 26, introducing several new features designed to enhance developer productivity. This update significantly integrates AI assistance, offering ChatGPT with GPT-5 (including a "Reasoning" model for complex tasks) as the new default, alongside support for Claude Sonnet 4. Developers now have the flexibility to choose their preferred chatbot, use API keys from other providers, or even run local AI models on Apple silicon Macs for coding help.
