Crescent library brings privacy to digital identity systems

Digital identities, though convenient, introduce significant privacy risks through "linkability," allowing various credential uses to be tracked and compiled into user profiles. To combat this, Microsoft Research has released Crescent, a cryptographic library designed to add "unlinkability" to common digital identity formats, including JSON Web Tokens and mobile driver's licenses. Crescent leverages zero-knowledge proofs, efficiently preventing any data beyond what a user explicitly chooses to reveal from linking their separate digital interactions, all without requiring updates to existing issuing systems.
