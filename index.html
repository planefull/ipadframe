BlueCodeAgent: A blue teaming agent enabled by automated red teaming for CodeGen AI

Large language models (LLMs) often generate insecure, biased, or malicious code, yet existing defensive "blue teaming" strategies face challenges like poor security concept alignment and over-conservatism. BlueCodeAgent, a new framework, addresses this by leveraging a diverse red-teaming process to generate knowledge, which is then distilled into actionable safety constitutions. These constitutions, combined with dynamic sandbox-based testing, enable BlueCodeAgent to significantly improve the detection of unsafe inputs and vulnerable code outputs, enhancing the trustworthiness and reliability of AI-generated code.
