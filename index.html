Ideas: More AI-resilient biosecurity with the Paraphrase Project

The Paraphrase Project, discussed in this Microsoft Research podcast, addresses the "dual-use" nature of AI in protein design, specifically how generative AI can be used to reformulate toxic proteins to evade existing biosecurity screening systems, creating a "zero-day" vulnerability. A cross-sector team successfully demonstrated this vulnerability using AI-rewritten versions of ricin and then developed and shared "patches" to strengthen these screening tools globally. This initiative not only enhanced AI-resilient biosecurity but also established a novel "perpetual gatekeeping" strategy for responsibly sharing sensitive research data, setting a precedent for managing information hazards in other dual-use fields.

iOS 26 Liquid Glass Design Copied by Android Smartphone Maker

Chinese smartphone maker Vivo has unveiled OriginOS 6, its latest operating system update, which closely mirrors Apple's "Liquid Glass" design language from iOS 26. The new interface features identical rounded buttons, translucent glass-like elements, squircle icons, and spatial scene wallpapers, making it easily mistaken for Apple's own OS. Despite Vivo's marketing efforts to describe the design with water-related themes, the clear inspiration from Apple's aesthetic is undeniable.
