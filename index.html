Crescent library brings privacy to digital identity systems

Digital identities, while convenient, pose significant privacy risks due to "linkability," where hidden identifiers allow separate credential uses to be associated and user profiles built. To address this, Microsoft Research has introduced Crescent, a cryptographic library that adds "unlinkability" to widely used identity formats like JSON Web Tokens and mobile driver's licenses. Crescent employs zero-knowledge proofs to enable selective disclosure of information without revealing linking data, ensuring user privacy without requiring credential issuers to update their systems, and is optimized for efficient mobile use.
