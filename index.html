Crescent library brings privacy to digital identity systems

Digital identities, though convenient, introduce privacy risks through "linkability," where separate uses of digital credentials can be associated with an individual for tracking. To address this, Microsoft Research developed Crescent, a cryptographic library that adds "unlinkability" to widely used identity formats like JSON Web Tokens and mobile driver's licenses. Crescent utilizes zero-knowledge proofs to allow users to prove specific facts without revealing linking identifiers, and it works without requiring issuing organizations to update their existing systems, making it efficient for mobile devices.

iOS 26: Import and Export Markdown Files in Apple Notes

In iOS 26, Apple Notes now supports Markdown, enabling users to seamlessly import and export files in this popular plain-text formatting language. This new feature allows developers, writers, and content creators to integrate their existing Markdown workflows with Apple's native note-taking app. Users can import Markdown files that convert to rich text in Notes, and also export their Notes content as Markdown for use in other applications.
