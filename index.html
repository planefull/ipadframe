BlueCodeAgent: A blue teaming agent enabled by automated red teaming for CodeGen AI

Large language models (LLMs) used for code generation introduce security risks like malicious code and vulnerabilities, but existing defensive "blue teaming" methods are often ineffective due to poor alignment with security concepts, over-conservatism, and incomplete risk coverage. Researchers developed BlueCodeAgent, an end-to-end blue teaming framework that addresses these challenges by integrating comprehensive red-teaming to generate diverse risk data and distill actionable safety "constitutions." This agent combines principled-level defense via these constitutions with nuanced-level analysis through dynamic sandbox-based testing, significantly improving the detection of biased instructions, malicious code, and vulnerabilities, and outperforming traditional prompting baselines.

Apple Invites App for iPhone Updated — Here's What's New

The latest update for the Apple Invites app on iPhone now allows guests to specify the number of adults and kids they are bringing to an event, in addition to bug fixes. The app, designed to simplify event invitations like birthday parties, lets users create custom invites with descriptions and shared photo albums, then share a link with guests. While creating events requires an iCloud+ subscription, anyone can RSVP for free via the iPhone app or the web.
