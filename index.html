Everything New in iOS 26.2 Beta 2

Apple has released iOS 26.2 beta 2 to developers, introducing several new features and design improvements to the operating system. Key updates include a "Liquid Glass" design for the Measure app's level, new game sorting options, CarPlay support for disabling pinned messages, and quicker menu animations. The beta also provides more information on Enhanced Safety Alerts for emergencies, which are enabled by default and can be refined with location data, with the full release expected in December.
