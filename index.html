Tell me when: Building agents that can wait, monitor, and act

Current LLM agents struggle with long-running monitoring tasks, failing because they don't know *when* to check for updates, leading to wasted resources or giving up prematurely. To solve this, Microsoft Research developed SentinelStep, a mechanism that enables agents to patiently monitor conditions for hours or days using dynamic polling and intelligent context management within a workflow. Implemented in their Magentic-UI prototype, SentinelStep significantly improves task reliability for longer durations, allowing agents to efficiently track things like emails, price drops, or repository changes without context overflow.

iOS 26.1 Features: Everything New in iOS 26.1

iOS 26.1, the first major update to iOS 26, is now available, bringing a range of new features, bug fixes, and user interface refinements. Key additions include a Liquid Glass transparency toggle, the option to disable the Lock Screen camera swipe, a new slide-to-stop for alarms, and expanded language support for Apple Intelligence and AirPods Live Translation. The update also introduces customizable workouts, various UI adjustments across several apps, improved background security, and for iPadOS 26.1, the reintroduction of Slide Over multitasking.
