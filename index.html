Crescent library brings privacy to digital identity systems

Digital identities, while convenient, create privacy risks through "linkability," allowing tracking of separate credential uses to build user profiles. To address this, Microsoft Research developed Crescent, a cryptographic library that adds "unlinkability" to common identity formats like JSON Web Tokens and mobile driver's licenses. Crescent utilizes zero-knowledge proofs with a two-stage process to efficiently prevent the linking of user interactions and enable selective disclosure of information, all without requiring existing systems to be updated.

Apple Invites App Now Offers Home Screen Widget

Apple has updated its Invites app to include new Home Screen widgets that allow users to count down to their next event, available in small, medium, and large sizes. These widgets can be easily added to the Home Screen, and tapping them opens the Invites app for more details. While the Invites app is free to download, an iCloud+ subscription is required to create new events.
