Crescent library brings privacy to digital identity systems

Digital identities, while convenient, pose privacy risks like "linkability," which allows separate uses of credentials to be tracked and user profiles built. To address this, Microsoft Research developed Crescent, a cryptographic library that adds "unlinkability" to common identity formats such as JSON Web Tokens and mobile driver's licenses. Utilizing zero-knowledge proofs, Crescent enables users to selectively disclose necessary information without revealing linking identifiers, functioning efficiently on mobile devices and without requiring issuers to update their systems.
