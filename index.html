BlueCodeAgent: A blue teaming agent enabled by automated red teaming for CodeGen AI

Large language models used for code generation introduce security risks like generating malicious, biased, or vulnerable code, but current defensive "blue teaming" approaches struggle with poor alignment to security concepts, over-conservatism, and incomplete risk coverage. To address this, researchers developed BlueCodeAgent, an end-to-end blue-teaming framework that leverages comprehensive "red teaming" to generate diverse risky data and distills it into actionable safety constitutions. This framework significantly improves defensive capabilities by combining principled-level defense via these constitutions with nuanced-level dynamic testing, enhancing models' understanding of risks and reducing false positives in vulnerability detection.

Apple Invites App for iPhone Updated — Here's What's New

The latest update to the Apple Invites app for iPhone now allows guests to specify the number of adults and children attending an event, alongside bug fixes and performance improvements. This app simplifies creating and sharing customizable event invitations, which can include descriptions, backgrounds, and shared photo albums. While creating events requires an iCloud+ subscription, anyone can RSVP for free via the iPhone app or the web.
