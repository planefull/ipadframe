BlueCodeAgent: A blue teaming agent enabled by automated red teaming for CodeGen AI

Large Language Models (LLMs) used for code generation introduce security risks like creating malicious, biased, or vulnerable code, which current defensive (blue-teaming) strategies struggle to address due to poor alignment with security concepts, over-conservatism, and incomplete risk coverage. To combat this, Microsoft Research and university partners developed BlueCodeAgent, an end-to-end blue-teaming framework that significantly enhances defenses by first using diverse red-teaming to generate comprehensive knowledge about risky code. This knowledge is then distilled into "constitutions" (actionable safety rules) for principled defense and combined with dynamic sandbox-based testing to accurately detect and prevent unsafe code, improving reliability and reducing false positives.

Apple Invites App for iPhone Updated — Here's What's New

The latest Apple Invites app update for iPhone introduces a new feature allowing guests to specify the number of adults and children they are bringing to an event. The app itself simplifies event invitations, enabling hosts to create detailed invites with custom backgrounds and shared photo albums, then share them via a link. While creating an event requires an iCloud+ subscription, anyone can RSVP for free through the iPhone app or on the web.
