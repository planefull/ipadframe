Crescent library brings privacy to digital identity systems

Digital identities offer convenience but present privacy risks through "linkability," allowing separate uses of credentials to be connected and user behavior tracked. To address this, Microsoft Research has released Crescent, a cryptographic library that introduces "unlinkability" to widely used identity formats like JSON Web Tokens and mobile driver's licenses. Crescent uses optimized zero-knowledge proofs to let users selectively prove facts about their credentials without revealing linking identifiers, protecting privacy without requiring updates from credential-issuing organizations.
