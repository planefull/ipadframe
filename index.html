Crescent library brings privacy to digital identity systems

Digital identities, while convenient, introduce privacy risks like "linkability," where hidden identifiers connect separate uses of credentials to create user profiles. To combat this, Microsoft Research developed Crescent, a cryptographic library that adds "unlinkability" to common digital identity formats such as JSON Web Tokens and mobile driver's licenses. Crescent leverages zero-knowledge proofs and a two-stage process to enable selective information disclosure without revealing linking data, effectively protecting user privacy without requiring extensive changes to existing identity infrastructure.
