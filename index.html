tvOS 26.2 Lets You Create Apple TV Profiles Without an Account, Adds Dedicated Kids Mode

With tvOS 26.2, Apple TV profiles can now be created without requiring an Apple Account, making it easier to set up guest or child profiles. This update introduces a dedicated kids' mode, allowing users to create child profiles by simply providing a name and setting customizable content ratings based on age. When a child's profile is active, the Apple TV app automatically filters its content to display only age-appropriate movies and shows, creating a truly restricted viewing experience.
