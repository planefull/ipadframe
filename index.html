Ideas: More AI-resilient biosecurity with the Paraphrase Project

The Paraphrase Project uncovered a critical "zero day" vulnerability where AI tools could redesign toxic protein sequences, such as ricin, to evade existing biosecurity screening systems. A collaborative team then developed red-teaming approaches and implemented "patches," resulting in a more robust, AI-resilient protein screening system that was shared with global gene synthesis companies. This effort not only strengthened biosecurity defenses but also established a novel perpetual gatekeeping strategy for sensitive scientific data, highlighting the ongoing need for proactive measures to manage AI's dual-use risks.
