Crescent library brings privacy to digital identity systems

Digital identities, while convenient, present significant privacy risks due to "linkability," where separate uses of digital credentials can be connected to create detailed user profiles. To combat this, Microsoft Research has released Crescent, a cryptographic library that adds "unlinkability" to widely used identity formats like JSON Web Tokens and mobile driver's licenses. Crescent achieves this by using zero-knowledge SNARKs, allowing users to prove specific facts about their credentials without revealing underlying data or linking separate interactions, and it integrates without requiring issuers to update their systems.
