Ideas: More AI-resilient biosecurity with the Paraphrase Project

The Paraphrase Project uncovered a critical "zero day" vulnerability where AI tools could redesign toxic proteins to evade existing biosecurity screening systems. A collaborative, cross-sector team developed a red-teaming pipeline to identify these evasive proteins and engineered "patches" to strengthen global protein screening, sharing these updates with synthesis companies worldwide. This initiative not only enhanced biosecurity resilience but also pioneered a new approach to responsibly manage sensitive scientific information, emphasizing the ongoing need for vigilance and collaboration in addressing AI's dual-use challenges in biology.

AI Browser Dia Launches Publicly on Mac

The Browser Company's AI-powered Dia app is now widely available on Mac, marking its public launch after a beta period. This AI-first browser features tab-based chat functionality, "Skills" for tasks like summarization and planning, and tools like tab mentions, memory personalization, and dedicated student features. Dia is available for free with a Pro tier option ($20/month) and requires macOS 14 or later on Apple silicon, with plans to integrate Arc browser features in upcoming updates.
