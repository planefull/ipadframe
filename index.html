Ideas: More AI-resilient biosecurity with the Paraphrase Project

The "Paraphrase Project" uncovered a "zero day" vulnerability where AI tools could redesign toxic protein sequences to bypass existing biosecurity screening systems, such as those used by DNA synthesis companies. A cross-sector team, using a cybersecurity-style response, successfully developed and deployed more robust screening methods to detect these AI-generated variants. This collaborative effort set a new precedent for addressing dual-use AI risks, demonstrating the importance of proactive security measures, continuous vigilance, and a novel approach to responsible information sharing in the evolving landscape of AI and synthetic biology.

Here's How the iOS 26.1 Transparency Toggle Changes Liquid Glass

iOS 26.1 beta 4 introduces a new "Tinted" option for Liquid Glass, allowing users to increase the opacity of UI elements and improve contrast. This setting, found under Display and Brightness, makes elements like Lock Screen notifications and in-app menus less transparent. While it enhances visibility in both light and dark modes, it primarily affects specific UI components and not the entire operating system like Control Center or the Home Screen.
