Crescent library brings privacy to digital identity systems

Digital identities, while convenient, create privacy risks through "linkability," allowing various digital interactions to be tracked and associated with individuals. Microsoft Research introduces Crescent, a cryptographic library designed to add "unlinkability" to widely used identity formats such as JSON Web Tokens and mobile driver's licenses. Crescent utilizes zero-knowledge proofs, optimized for mobile devices, to enable users to selectively disclose information without revealing data that could link their separate activities, effectively protecting privacy without requiring changes to issuing organizations' systems.
