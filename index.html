Crescent library brings privacy to digital identity systems

Digital identities create privacy risks through "linkability," where hidden identifiers can track and link a user's separate activities across different digital credential uses. To address this, Microsoft Research has developed Crescent, a cryptographic library that adds "unlinkability" to widely used identity formats like JSON Web Tokens and mobile driver's licenses. Crescent utilizes zero-knowledge proofs and a two-stage process, enabling users to selectively disclose information while preventing any part of the credential, beyond explicit choices, from linking their interactions, all without requiring issuers to update their systems.
