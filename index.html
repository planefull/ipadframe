Crescent library brings privacy to digital identity systems

Digital identities, while convenient, pose significant privacy risks due to "linkability," which allows various uses of credentials to be connected, leading to extensive user tracking. To counter this, Microsoft Research has introduced Crescent, a cryptographic library that adds "unlinkability" to widely used digital identity formats like JSON Web Tokens and mobile driver's licenses. Crescent leverages zero-knowledge proofs, optimized for mobile devices, to prevent separate digital interactions from being linked, thereby enhancing user privacy without requiring credential issuers to update their existing systems.

How Do Case Makers Reveal iPhone 17 Designs Before Apple?
