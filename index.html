Crescent library brings privacy to digital identity systems

Digital identities offer convenience but create privacy risks through "linkability," which allows separate credential uses to be associated with a person. To address this, Microsoft Research developed Crescent, a cryptographic library that adds "unlinkability" to common digital identity formats like JSON Web Tokens and mobile driver's licenses. Crescent utilizes zero-knowledge SNARKs and a two-stage proof generation process, enabling users to selectively disclose information (e.g., proving age without revealing the exact date) without their interactions being linked, all while working with existing identity infrastructure.
