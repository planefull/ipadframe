Crescent library brings privacy to digital identity systems

Digital identities, while convenient, introduce privacy risks like "linkability," where hidden identifiers can connect separate credential uses to build detailed user profiles. To address this, Microsoft Research has released Crescent, a cryptographic library that adds "unlinkability" to common identity formats such as JSON Web Tokens and mobile driver's licenses. Crescent leverages zero-knowledge proofs with preprocessing to enable users to selectively disclose information and prevent their digital interactions from being linked, all without requiring changes to existing issuing systems.

AirPods Pro 3 Said to Gain Two Health Sensors, But Translation Delayed

Apple's upcoming AirPods Pro 3 are reportedly set to include in-ear temperature and heart-rate sensing, according to an anonymous tip received by 9to5Mac.
