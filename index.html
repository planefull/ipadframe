Crescent library brings privacy to digital identity systems

Digital identities introduce privacy risks through "linkability," allowing separate credential uses to be tracked and user profiles to be built. To combat this, Microsoft Research has developed Crescent, a cryptographic library that adds "unlinkability" to common identity formats like JSON Web Tokens and mobile driver's licenses. Crescent utilizes efficient zero-knowledge proofs to enable users to selectively disclose information and prove facts without revealing unnecessary data or allowing their digital interactions to be connected, all without requiring updates to credential issuers' systems.
