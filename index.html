Crescent library brings privacy to digital identity systems

Digital identities offer convenience but introduce privacy risks, particularly "linkability," where hidden identifiers connect various user activities to build detailed profiles. To address this, Microsoft Research developed Crescent, a cryptographic library that brings "unlinkability" to widely used identity formats such as JSON Web Tokens and mobile driver's licenses. Crescent uses zero-knowledge proofs, optimized for mobile devices, to allow users to selectively reveal information without exposing linking identifiers, enhancing privacy without requiring credential issuers to update their systems.
