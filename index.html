Crescent library brings privacy to digital identity systems

Digital identities offer convenience but introduce privacy risks like "linkability," where hidden identifiers can connect a user's separate activities to build detailed profiles. To address this, Microsoft Research developed Crescent, a cryptographic library that adds "unlinkability" to widely used identity formats like JSON Web Tokens and mobile driver's licenses. Crescent achieves this by using efficient zero-knowledge proofs, allowing users to selectively disclose information without revealing linking identifiers, and integrates with existing systems without requiring major infrastructure changes.
