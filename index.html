BlueCodeAgent: A blue teaming agent enabled by automated red teaming for CodeGen AI

Large Language Models (LLMs) used for code generation pose security risks, including producing malicious, biased, or vulnerable code, with current defensive (blue teaming) methods facing significant limitations. To address this, Microsoft Research developed BlueCodeAgent, an end-to-end blue teaming framework that leverages automated red teaming to generate diverse attack data and distill it into actionable safety "constitutions." This knowledge-enhanced approach, combined with dynamic testing, significantly improves LLMs' ability to detect and prevent unsafe code generation by providing concrete guidelines and reducing false positives, thereby enhancing trust and reliability in the broader software ecosystem.

Apple Invites App for iPhone Updated — Here's What's New

The latest update to the Apple Invites app for iPhone now allows guests to specify the number of adults and kids they are bringing to an event. The app itself simplifies event invitations by letting hosts create customized invites with descriptions and shared photo albums, then share a link with guests. While creating events requires an iCloud+ subscription, anyone can RSVP for free via the iPhone app or on the web.
