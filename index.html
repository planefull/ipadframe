Ideas: More AI-resilient biosecurity with the Paraphrase Project

The Microsoft Research Podcast discusses the "Paraphrase Project," which uncovered a critical biosecurity vulnerability where AI tools could redesign toxic proteins, like ricin, to bypass existing DNA synthesis screening systems. A cross-sector team then collaborated to develop and implement "patches"—more robust protein screening methods—and shared them globally to strengthen biosecurity defenses against these AI-generated sequences. This initiative demonstrated a "zero-day" vulnerability in AI biosecurity and established a new model for responsible disclosure and ongoing international cooperation to manage the dual-use risks of emerging AI capabilities.

Must-Try tvOS 26 Features

Apple's tvOS 26 update for the Apple TV introduces several useful enhancements aimed at improving user experience and personalization. Noteworthy new features include much easier profile switching for tailored content recommendations, an overhauled Apple TV app displaying more shows, and the ability to set third-party AirPlay speakers as default for convenience. Additionally, users can now leverage their iPhone as a microphone for Apple Music Sing karaoke and benefit from an updated FaceTime interface with Live Translation.
