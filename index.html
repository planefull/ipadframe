Crescent library brings privacy to digital identity systems

Digital identities, though convenient, introduce privacy risks through "linkability," where separate uses of digital credentials can be tracked to build user profiles. To address this, Microsoft Research has released Crescent, a cryptographic library designed to add "unlinkability" to common identity formats like JSON Web Tokens and mobile driver's licenses. Crescent uses zero-knowledge proofs to allow users to selectively disclose information without revealing identifiable data or linking separate interactions, and it works without requiring issuing organizations to update their systems.
