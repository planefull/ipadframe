Ideas: More AI-resilient biosecurity with the Paraphrase Project

The Microsoft Research *Ideas* podcast episode discusses the "Paraphrase Project," which uncovered a significant vulnerability: AI tools could redesign toxic proteins to evade existing biosecurity screening systems, posing a "zero day" threat. A cross-sector team, including Microsoft, Twist Bioscience, and IBBIS, addressed this by developing AI red-teaming approaches to test vulnerabilities, create more robust screening systems, and implement "patches" shared globally. This collaborative effort, detailed in *Science*, not only strengthened biosecurity against AI misuse but also pioneered a new perpetual gatekeeping strategy for handling sensitive dual-use research data responsibly.
