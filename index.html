RedCodeAgent: Automatic red-teaming agent against diverse code agents

RedCodeAgent is the first fully automated and adaptive red-teaming agent designed to evaluate the safety of large language model (LLM)-based code agents, addressing the shortcomings of traditional static safety benchmarks. It operates by combining an adaptive memory module, a specialized toolbox including code substitution, and simulated execution environments to continuously learn and refine attack strategies. This innovative approach effectively uncovers diverse vulnerabilities across multiple programming languages and code agents, outperforming existing methods in attack success rates and discovering previously unknown weaknesses.

Delta Now Offers iOS 26's Upgraded Boarding Passes in Apple Wallet

Delta Airlines has reportedly added support for iOS 26's enhanced boarding pass features within the Apple Wallet app. These upgraded boarding passes offer Live Activities for real-time flight tracking on the Lock Screen and Dynamic Island, along with quick access to Apple Maps for airport directions and Find My for baggage tracking. Delta joins United and several other airlines expected to offer these convenient functionalities, which also include automatic updates for flight details.
