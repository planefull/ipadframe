Crescent library brings privacy to digital identity systems

Digital identities, while convenient, introduce significant privacy risks, particularly "linkability," where hidden identifiers connect separate credential uses to create detailed user profiles. To address this, Microsoft Research has released Crescent, a cryptographic library designed to add "unlinkability" to widely used identity formats like JSON Web Tokens and mobile driver's licenses. Crescent leverages zero-knowledge proofs to allow users to selectively disclose information without revealing linking identifiers or requiring issuing organizations to update their systems, thereby enhancing privacy.

Apple Preparing iOS 18.7 for iPhones as iOS 26 Release Date Nears

Apple is preparing to release iOS 18.7, expected in September, primarily to deliver security fixes to compatible iPhones. This update will be among the last for older devices such as the iPhone XS, XS Max, and XR, as they will not support the upcoming iOS 26. iOS 26, which is compatible with iPhone 11 and newer, is also nearing its public release around mid-September.
