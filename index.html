Ideas: More AI-resilient biosecurity with the Paraphrase Project

The Paraphrase Project, led by Microsoft's Chief Scientific Officer Eric Horvitz, revealed a "zero day" vulnerability where AI tools could redesign toxic proteins, like ricin, to bypass existing biosecurity screening systems. Employing a cybersecurity-style response, the cross-sector team developed and shared "patches" that significantly improved the detection capabilities of these screening tools for AI-generated variants across a range of hazardous proteins. This collaborative effort established a more robust biosecurity framework and pioneered a "perpetual gatekeeping" strategy for responsible data sharing, offering a model for managing dual-use risks in emerging technologies.

More Apple Apps Get Liquid Glass Redesign

Apple has updated its TestFlight and Apple Support apps, integrating new Liquid Glass designs to match the aesthetic of iOS 26. This redesign brings elements like rounded buttons, floating navigation bars, translucency, and layered app icons to both applications. TestFlight also received an icon overhaul, significant accessibility improvements including VoiceOver and Larger Text, and a new Tester Matching feature to help users find beta apps.
