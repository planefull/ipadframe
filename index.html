Crescent library brings privacy to digital identity systems

Digital identities, while convenient, create significant privacy risks due to "linkability," where separate uses of credentials can be tracked to build detailed user profiles. To combat this, Microsoft Research has developed Crescent, a cryptographic library that adds "unlinkability" to widely used identity formats like JSON Web Tokens and mobile driver's licenses. Crescent utilizes zero-knowledge proofs, optimizing them for mobile devices, to enable users to selectively disclose information without revealing underlying data or allowing their interactions to be linked, thus enhancing privacy without requiring issuers to update their systems.
