Crescent library brings privacy to digital identity systems

Digital identities, though convenient, create privacy risks through "linkability," where hidden identifiers allow tracking and profiling across various credential uses like mobile driver's licenses and app logins. To address this, Microsoft Research has developed Crescent, a cryptographic library that adds "unlinkability" to common identity formats such as JSON Web Tokens and mobile driver's licenses. Crescent uses efficient zero-knowledge proofs to enable selective disclosure and prevent unrelated information from linking a user's separate digital interactions, all without requiring existing credential issuers to update their systems.
