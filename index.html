Tell me when: Building agents that can wait, monitor, and act

Modern LLM agents struggle with long-running monitoring tasks, such as tracking emails or price changes, because they lack the ability to know *when* to check, leading to inefficiency or failure. To solve this, Microsoft Research introduces SentinelStep, a mechanism implemented in Magentic-UI that enables agents to perform these tasks by using dynamic polling and intelligent context management. This approach significantly improves agent reliability for extended monitoring periods, laying the groundwork for more patient, proactive, and resource-efficient AI assistants.

iOS 26.1 Features: Everything New in iOS 26.1

iOS 26.1 is now available as the first major update to iOS 26, introducing numerous new features and addressing user complaints. Key improvements include a new Liquid Glass transparency toggle, the ability to disable the Lock Screen camera swipe, and an updated slide-to-stop interface for alarms and timers. Additionally, the update expands Apple Intelligence and AirPods Live Translation to more languages, offers custom workout creation in the Fitness app, and brings various UI refinements, security enhancements, and the reintroduction of Slide Over for iPadOS 26.1.
