Tell me when: Building agents that can wait, monitor, and act

Large Language Model (LLM) agents currently struggle with long-running monitoring tasks, failing because they don't know *when* to check for updates, leading to resource waste or task abandonment. To address this, Microsoft Research introduced SentinelStep, a mechanism implemented in Magentic-UI that wraps agents in a workflow with dynamic polling and smart context management. This approach allows agents to monitor conditions for extended periods by adjusting polling frequency and preserving state, significantly improving reliability for tasks lasting hours.
