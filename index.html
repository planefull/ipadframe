Crescent library brings privacy to digital identity systems

Digital identities offer convenience but pose privacy risks, particularly "linkability," where hidden identifiers can connect separate credential uses to create user profiles. To combat this, Microsoft Research introduced Crescent, a cryptographic library that adds unlinkability to common digital identity formats like JSON Web Tokens and mobile driver's licenses. Crescent utilizes zero-knowledge proofs and a two-stage process to allow users to selectively disclose information and prove facts about their credentials without revealing personal data or enabling tracking of their interactions, all without requiring system updates from credential issuers.
