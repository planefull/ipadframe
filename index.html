RedCodeAgent: Automatic red-teaming agent against diverse code agents

Code agents introduce critical safety risks that existing static benchmarks and red-teaming methods often fail to detect, especially concerning code execution and combined jailbreak tools. To address this, researchers developed RedCodeAgent, the first fully automated and adaptive red-teaming agent designed to evaluate the safety of LLM-based code agents. RedCodeAgent uses adaptive memory, specialized tools, and simulated execution environments to achieve higher attack success rates, uncover new vulnerabilities, and outperform other methods across diverse programming languages and code agents.
