BlueCodeAgent: A blue teaming agent enabled by automated red teaming for CodeGen AI

BlueCodeAgent is a novel blue-teaming framework designed to enhance the security and trustworthiness of large language models (LLMs) used for code generation, addressing issues like malicious, biased, or vulnerable code output. It achieves this by employing a comprehensive automated red-teaming process to generate diverse risky data, which is then distilled into actionable safety "constitutions" for principled defense. Additionally, BlueCodeAgent utilizes dynamic sandbox-based testing to validate potential vulnerabilities and mitigate over-conservatism, resulting in an average 12.7% improvement in F1 score across various risk detection tasks.
