Crescent library brings privacy to digital identity systems

Digital identities, though convenient, introduce significant privacy risks through "linkability," allowing hidden identifiers to connect separate credential uses for tracking and profiling. To address this, Microsoft Research developed Crescent, a cryptographic library that adds "unlinkability" to common identity formats like JSON Web Tokens and mobile driver's licenses. Crescent utilizes efficient zero-knowledge proofs, optimized for mobile devices, to ensure users can selectively disclose information without revealing any data that could link their digital interactions, all without requiring updates to existing issuer systems.
