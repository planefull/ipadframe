Crescent library brings privacy to digital identity systems

Digital identities offer convenience but pose privacy risks due to "linkability," where hidden identifiers allow tracking and profiling of user activities across various digital interactions. Microsoft Research has developed Crescent, a cryptographic library designed to add unlinkability to common identity formats like JSON Web Tokens and mobile driver's licenses. Crescent achieves this through efficient zero-knowledge proofs, optimized for mobile devices, allowing users to selectively disclose information without revealing linking identifiers or requiring updates to existing credential issuance systems.
