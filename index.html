Crescent library brings privacy to digital identity systems

Digital identities, though convenient, create significant privacy risks due to "linkability," which allows various credential uses to be associated with an individual, enabling extensive tracking and profiling. To combat this, Microsoft Research has released Crescent, a cryptographic library designed to add "unlinkability" to widely used identity formats like JSON Web Tokens and mobile driver's licenses. Crescent leverages zero-knowledge proofs, optimized for mobile devices, to enable users to selectively disclose information without revealing linking identifiers, thereby protecting privacy without requiring major infrastructure updates from credential issuers.
