Crescent library brings privacy to digital identity systems

Digital identities, while convenient, pose privacy risks due to "linkability," where hidden identifiers can track and connect a user's separate online activities. To address this, Microsoft Research has released Crescent, a cryptographic library designed to add "unlinkability" to widely used identity formats, including JSON Web Tokens and mobile driver's licenses. Crescent leverages zero-knowledge proofs and an efficient two-stage process to prevent credential linking, ensuring users only reveal explicitly chosen information without requiring changes to existing credential issuance systems.
