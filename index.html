Crescent library brings privacy to digital identity systems

Digital identities offer convenience but create privacy risks through "linkability," where separate uses of credentials can be tracked to build user profiles. To address this, Microsoft Research has released Crescent, a cryptographic library that adds "unlinkability" to widely used identity formats like JSON Web Tokens and mobile driver's licenses. Crescent uses efficient zero-knowledge proofs to allow users to selectively disclose information and prevent tracking across different interactions, without requiring existing credential issuance systems to be updated.
