Crescent library brings privacy to digital identity systems

Digital identities, though convenient, introduce privacy risks like "linkability," where various uses of credentials can be connected to build user profiles. To address this, Microsoft Research has developed Crescent, a cryptographic library that adds "unlinkability" to common digital identity formats, including JSON Web Tokens and mobile driver's licenses. Crescent achieves this by using zero-knowledge proofs, optimized for mobile devices through preprocessing, allowing users to prove facts without revealing unnecessary or linkable data, and without requiring credential issuers to update their existing systems.
