MMCTAgent: Enabling multimodal reasoning over large video and image collections

Modern multimodal AI models struggle with analyzing long-form video and large image collections due to limitations in temporal reasoning and iterative refinement. To address this, Microsoft Research developed MMCTAgent, a Multi-modal Critical Thinking Agent built on AutoGen's multi-agent system. MMCTAgent uses a Planner-Critic architecture with specialized agents (ImageAgent and VideoAgent) to perform structured, iterative reasoning, tool-based analysis, and self-evaluation, significantly enhancing performance on complex visual understanding tasks.

iOS 26.2 Will Bring 10+ New Features to Your iPhone

Apple is set to release iOS 26.2 around mid-December, introducing significant regional changes for iPhone users in Japan, along with a variety of new features for everyone. Notable additions include an alarm option for the Reminders app, a customizable Liquid Glass clock on the Lock Screen, and enhanced AirDrop functionality allowing temporary file sharing via one-time codes. Furthermore, users in Japan will gain the ability to choose a different default voice assistant, select a preferred search engine, and install third-party app stores, while AirPods Live Translation expands to the EU and App Store compliance tools are added for Texas.
