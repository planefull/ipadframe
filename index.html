Crescent library brings privacy to digital identity systems

Digital identities, while offering convenience, create privacy risks due to "linkability," where hidden identifiers in credentials can track user activities across different applications and services. To address this, Microsoft Research has released Crescent, a cryptographic library that adds "unlinkability" to widely used identity formats like JSON Web Tokens and mobile driver's licenses. Crescent leverages zero-knowledge proofs to allow users to verify specific facts about their credentials without revealing underlying data or connecting separate interactions, making it privacy-preserving and efficient for mobile devices without requiring system updates from credential issuers.
